# Extensions

InForm-PE uses various libraries, all of which are available for developers to include in their InForm-PE projects. These libraries reside in the `InForm/extensions` directory.

* [**QBDS**](./QBDS.md): Generic data structure libraries.
* [**Algo**](./Algo.md): General-purpose algorithms.
* [**Catch**](./Catch.md): Minimalistic test framework.
* [**FontMgr**](./FontMgr.md): Finds and retrieves information about system-installed fonts.
* [**GIFPlay**](./GIFPlay.md): Plays animated GIF files.
* [**INI-Manager**](./INI-Manager.md): Reads and writes INI files.
* [**Memory**](./Memory.md): Low-level memory operations.
* [**Pathname**](./Pathname.md): Parses filenames and sanitizes paths.
* [**Resource**](./Resource.md): Loads Base64-encoded binary resources.
* [**String**](./String.md): Classifies characters and formats strings.
* [**StringFile**](./StringFile.md): Treats a STRING as a binary file for in-memory I/O.
* [**Time**](./Time.md): High-resolution timer and frequency measurement.
